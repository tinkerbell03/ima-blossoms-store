Ima Blossoms | Handcrafted Everlasting Flowers & Gifts 🌸

Ima Blossoms is a premium digital storefront for a Sri Lankan-based floral business. We specialize in handcrafted everlasting blooms—including our signature glitter roses—custom gift hampers, and elegant hair accessories. This platform is designed to provide a seamless, high-performance shopping experience for customers looking for "sparkle that never fades."
✨ Key Features

    📱 Fully Responsive Design: Optimized for a flawless experience across all devices, from mobile phones and tablets to large desktop screens.

    ⚡ High Performance: Built with Bun and Vite for near-instant load times and a smooth user interface.

    🔍 SEO Optimized: Includes a comprehensive suite of Meta tags, Open Graph (OG) tags for social media, and a clean document structure to ensure maximum visibility on Google and social platforms.

    🛍️ Product Showcases: Interactive sections for Glitter Roses, Custom Gift Boxes, and Satin Accessories.

    🎨 Modern UI/UX: A clean, elegant aesthetic using shadcn/ui and Radix UI primitives for accessible and professional interactions.

🛠️ Techniques & Languages Used

This project leverages a modern "bleeding-edge" tech stack to ensure scalability and speed:
Core Languages & Runtime

    TypeScript: Used for type-safety and robust code management.

    React 19: The latest version of the world's most popular library for building user interfaces.

    Bun: A fast all-in-one JavaScript runtime and package manager used for development and building.

Styling & UI

    Tailwind CSS: A utility-first CSS framework for rapid, custom styling.

    shadcn/ui: High-quality, accessible UI components built on top of Radix UI.

    Lucide React: A beautiful and consistent icon library.

Development Tools

    Vite: The frontend build tool providing an extremely fast development environment.

    ESLint: Ensuring code quality and consistency across the project.
